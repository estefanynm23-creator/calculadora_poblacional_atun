# Tuna Population Calculator
# Calculadora poblacional del atún 

## Overview
The Tuna Population Calculator is an interactive application designed to model and analyze the population growth of tuna using the Beverton-Holt model. This project aims to provide researchers and biologists with a tool to simulate tuna population dynamics and analyze biological data through numerical curve fitting methods.
## Resumen 
El Calculador de Población de Atún es una aplicación interactiva diseñada para modelar y analizar el crecimiento de la población de atún utilizando el modelo de Beverton-Holt. Este proyecto tiene como objetivo proporcionar a investigadores y biólogos una herramienta para simular la dinámica poblacional del atún y analizar datos biológicos mediante métodos numéricos de ajuste de curvas.

## Features
- **Interactive GUI**: A user-friendly graphical interface for inputting parameters and visualizing population growth.
- **Beverton-Holt Model**: Implements the Beverton-Holt model for predicting future populations based on current data.
- **Numerical Curve Fitting**: Includes methods for accurately fitting biological data to improve model predictions.
- **Data Visualization**: Generates plots to visualize population dynamics over time.
- **Sample Data**: Comes with sample biological data for testing and analysis.
## Caracteristicas
- **Interfaz gráfica interactiva**: Una interfaz gráfica fácil de usar para ingresar parámetros y visualizar el crecimiento de la población.
- **Modelo Beverton-Holt**: Implementa el modelo Beverton-Holt para predecir poblaciones futuras basadas en datos actuales.
- **Ajuste numérico de curvas**: Incluye métodos para ajustar con precisión los datos biológicos y mejorar las predicciones del modelo.
- **Visualización de datos**: Genera gráficos para visualizar la dinámica de la población a lo largo del tiempo.
- **Datos de muestra**: Viene con datos biológicos de muestra para pruebas y análisis.

